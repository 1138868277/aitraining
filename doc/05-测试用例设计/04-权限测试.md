# 权限测试

## 说明

当前版本**未接入用户认证与权限管理系统**，所有用户具有相同的操作权限。应用部署在内网环境，通过网络层进行访问控制。

## 访问控制

| 编号 | 场景 | 前置条件 | 步骤 | 预期结果 |
|------|------|---------|------|---------|
| AUTH-01 | 无需登录即可访问页面 | 前后端服务已启动 | 直接访问任何一个页面路由 | 页面正常加载，无登录拦截 |
| AUTH-02 | 所有接口无需认证 | 后端服务已启动 | 调用任意 API 接口 | 接口正常返回数据，无 401 鉴权错误 |
| AUTH-03 | CORS 跨域限制 | 后端服务已启动 | 从非配置的 FRONTEND_URL 发起 API 请求 | 被 CORS 策略拦截 |
| AUTH-04 | 数据源配置锁定 | 系统设置页面已加载 | 查看数据源管理表单 | 表单默认锁定，不可直接编辑 |
| AUTH-05 | 数据源配置解锁 | 表单锁定状态 | 点击解锁按钮 | 表单变为可编辑状态 |

## OAuth 预留（当前未启用）

环境变量中已配置 OAuth 参数（`OAUTH_CLIENT_ID`、`OAUTH_CLIENT_SECRET`、`AUTH_BASE_URL`），但当前版本未实现 OAuth 登录流程，以上参数为后续接入统一认证预留。

## 安全测试检查项

| 编号 | 场景 | 前置条件 | 步骤 | 预期结果 |
|------|------|---------|------|---------|
| AUTH-06 | SQL 注入防护 | 后端服务已启动 | 在 API 参数中传入恶意 SQL（如 `' OR 1=1 --`） | 参数化查询拦截，不返回异常数据 |
| AUTH-07 | 文件上传类型校验 | 后端服务已启动 | 上传非 .xlsx/.xls 文件（如 .exe、.html） | 被拒绝，提示"文件格式不合法" |
| AUTH-08 | 文件上传大小限制 | 后端服务已启动 | 上传超过限制大小的文件 | 被拒绝，提示"文件大小超出限制" |
| AUTH-09 | 错误信息不泄露内部细节 | 后端服务已启动 | 触发系统异常（如数据库连接断开） | 返回通用错误提示，不包含堆栈或 SQL 信息 |
