# 技术选型

## 技术栈

- 语言：TypeScript（前端）、Java 17（后端）
- 前端框架：Vue 3 + Element Plus + Vite
- 后端框架：Spring Boot + MyBatis-Plus
- 数据库：PostgreSQL (SeaboxSQL)
- 测试工具：Vitest + JUnit 5

## 选型理由

- Node.js：作为前端开发和构建运行时，支持 Vite 快速热更新，生态成熟，与 TypeScript 深度集成。
- 包管理器：使用 pnpm。相比 npm/yarn，节省磁盘空间、安装速度快、天然支持 monorepo（未来可扩展）。
- 其他依赖：
    Zod：前端表单校验与运行时类型验证，保证输入数据安全。
    Element Plus：Vue 3 企业级组件库，满足后台管理类页面快速搭建。
    MyBatis-Plus：简化数据库访问，避免手写大量 SQL，同时保留灵活性。
    EasyExcel：处理 Excel 批量导入导出，内存友好。
    公司 Auth/Base 服务：统一登录、鉴权、文件存储，减少重复建设。

## 版本要求

- Node.js：≥ 24.0
- 包管理器：pnpm ≥ 9.0
- 其他依赖：Java 17、Spring Boot 2.7、PostgreSQL 14
