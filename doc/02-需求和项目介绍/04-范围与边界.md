# 范围与边界

## 本期功能范围

### 功能模块

#### 1. 编码生成（code-generation）
- **编码生成**：根据 11 类字典筛选条件实时生成合规编码，支持单条生成和批量生成（多维度笛卡尔积组合）
- **草稿管理（临时区）**：生成的编码可保存至临时草稿区，支持查看草稿列表、单条删除、批量清空
- **最近筛选条件**：自动记录最近使用的筛选条件，支持快速复用
- **历史记录管理**：生成的编码可入库保存，支持历史记录的分页查看、单条删除、批量删除
- **Excel 导出**：支持编码生成结果和历史记录的 Excel 导出
- **快速检索面板**：在生成页面内嵌数据码模糊搜索，支持类型筛选（风电 F/光伏 G/水电 S）与锁定功能
- **新增数据码**：在生成页面内可直接新增缺失的数据码字典项

#### 2. 自动编码匹配（auto-code）
- **Excel 上传**：支持上传含设备名称、场站、二级类码等字段的 Excel 文件
- **列映射向导**：将 Excel 列映射到系统字段（场站名称、二级类码、三级类码、数据类码、数据码、测点名称等）
- **编码配置**：支持配置项目期号、并网线路、前缀号、一级类码等固定参数
- **自动字典匹配与生成**：基于映射后的字段自动匹配字典编码，批量生成 31 位编码
- **数量限制**：单次最多处理 20000 条编码

#### 3. 编码解析（code-parse）
- **单条解析**：输入 31 位编码，按 11 个段位拆解并反向解析字典含义
- **批量解析**：粘贴多条编码，批量解析段位含义，表格展示并支持展开查看详细段位
- **段位可视化展示**：分块卡片展示各段位编码与含义，编码结构图彩色分段展示
- **识别状态标记**：全部识别/部分未识别状态提示，未识别段位标红
- **批量解析结果导出**：将批量解析结果导出为 Excel

#### 4. 重复编码稽核（code-audit）
- **批量粘贴稽核**：粘贴编码列表，系统与全量测点数据比对
- **稽核结果展示**：逐条标记"重复"/"不重复"，附带说明
- **数量限制**：单次稽核上限 10000 条

#### 5. 编码修正（code-correction）
- **Excel 导入修正数据**：上传含"测点编码、测点描述、修改内容"三列的文件
- **批量修正执行**：按修改内容解析需要修改的段位，批量生成修正后编码
- **分段对比展示**：修正前后编码按 11 个段位逐段对比，修改部分高亮标记（旧值标红删除线，新值标绿加粗）
- **重复自动检测**：修正后编码自动与全量测点比对，标记重复风险
- **分批处理与进度跟踪**：超过 50 条自动分批处理，实时进度百分比显示
- **修正结果导出**：修正结果导出为 Excel，含新旧编码、修改内容、重复校验结果
- **数量限制**：单次修正上限 100000 条

#### 6. 字典管理（dict）
- **字典树**：分层树形结构展示全部 11 类字典项，前三级缓存加载，数据码层懒加载
- **数据码查询**：关键字模糊搜索字典项，支持类型筛选
- **树形浏览**：按层级导航浏览字典结构
- **字典项概览**：按风电/光伏/水电类型统计各字典维度数据量
- **手动新增字典**：支持手动新增和批量新增字典项，标记为手工添加
- **数据类码筛选**：按数据类型获取对应的数据类码
- **二级类码筛选**：按设备类型（风电/光伏/水电）获取对应的二级类码
- **新增字典记录**：手动新增字典记录的分页查询，支持类型和二级类码筛选、Excel 导出

#### 7. 字典审批（approval）
- **区域提交申请**：提交数据码/数据类码新增申请（含类型、二级类码、数据类码、数据码、名称）
- **集团审批处理**：审核通过或驳回（需填写原因）
- **审批列表查询**：待审批/已审批列表分页展示，支持按区域和状态筛选

#### 8. 场站管理（station）
- **场站列表**：分页查看场站信息
- **新增场站**：单个新增场站
- **批量新增**：批量导入新增场站
- **编辑场站**：修改场站信息
- **删除场站**：单个删除和全部清空（含确认提示）
- **Excel 导出**：场站列表导出

#### 9. 统计分析（statistics）
- **编码生成统计**：
  - 概览卡片：累计生成总数、今日/本周/本月生成数
  - 类型维度：风电、光伏、水电、不区分类型的编码数量统计
  - 二级类码维度：按设备类型筛选，横向柱状图展示
  - 场站维度：按场站聚合的横向柱状图
  - 分组详情列表：按生成批次/分组展示，支持展开查看单条编码、多选导出、批量删除
  - 生成趋势：编码生成量的时间趋势
- **全量测点统计**：
  - Excel 导入：上传存量测点编码文件，支持进度条与预估剩余时间，可取消导入
  - 概览卡片：总测点数、风电/光伏/其他分类统计
  - 二级类码维度：按设备类型筛选的横向柱状图
  - 场站维度：按场站聚合的横向柱状图
  - 数据清空：一键清空所有导入的测点数据

#### 10. 时序稽核规则（time-series-rules / tsr）
- **数据导入**：导入场站数据和测点数据（Excel 格式，支持 200MB 以内文件）
- **规则生成**：自动生成死值（sz）、突变（tb）、越限（yx）、终端（zd）四类稽核规则
- **进度跟踪**：分步骤流水线可视化，实时展示导入和生成进度
- **数据概览**：统计已导入的场站数和测点数
- **单类型 Excel 导出**：按规则类型分别导出为 Excel
- **整体 ZIP 导出**：全部 4 种规则类型打包为一个 ZIP
- **分批拆分导出**：按 Z7 拆分逻辑，导出分批后的规则文件 ZIP
- **总体导出（分类型+分批）**：总体结果单文件或分批 ZIP
- **数据清空**：一键清空所有导入的场站和测点数据
- **数据库连接测试**：测试当前租户 TSR 相关数据库连接

#### 11. 数据源管理（datasource）
- **配置查看**：查看当前 PostgreSQL 数据源连接配置
- **连接测试**：测试配置的连接是否可用
- **保存切换**：保存新配置并在运行时动态切换数据源，无需重启服务
- **锁定保护**：配置表单默认锁定，防止误修改

#### 12. 用户认证与权限管理（auth）
- **登录认证**：用户名+密码登录，JWT Token 鉴权，24 小时有效期
- **当前用户信息**：根据 Token 获取当前用户信息（用户名、显示名、区域、租户）
- **用户列表**：集团管理员查看所有用户
- **新增用户**：集团管理员新增用户（用户名、显示名、区域、密码）
- **编辑用户**：集团管理员修改用户信息
- **删除用户**：集团管理员删除用户
- **默认用户**：admin（集团管理员）/ yunnan（云南区域）/ fujian（福建区域）

## 非本期范围

- 操作审计日志：编码操作未记录操作人信息（用户认证已实现但未记录操作日志）
- 编码规则可视化配置：编码规则为固定算法，未提供前端规则配置界面
- 字典树编辑：字典树为只读展示，字典项通过手动新增接口或审批流程添加
- 移动端适配：当前仅支持桌面端浏览器
- 审批流程前端页面：审批管理后台 API 已实现，前端管理页面为独立组件未集成至导航

## 边界说明

- 技术栈：Node.js/Express（TypeScript）后端 + Vue 3（Element Plus）前端 + PostgreSQL 数据库
- 多租户：支持 admin（集团）/ yunnan（云南）/ fujian（福建）三个租户，数据通过 Schema 隔离
- 临时数据：草稿数据存储于后端内存，服务重启后丢失；校验任务数据存储于后端内存
- 持久化数据：历史编码记录、字典数据、场站数据、统计导入数据、用户数据持久化于 PostgreSQL
- 数据源配置：持久化于 `config/datasource.json` 文件，修改后运行时生效
- 用户认证：用户数据存储于 admin 租户的 `cec_sys_user` 表，Token 签发使用 HS256 算法
