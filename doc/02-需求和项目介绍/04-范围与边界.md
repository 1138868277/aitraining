# 范围与边界

## 本期范围

- 必做功能：
    1.编码生成：11 项字典筛选、实时编码生成、临时保存 / 批量汇总、复制 / Excel 导出、生成记录入库与追溯
    2.编码校验：单条 / 批量 / Excel 导入、合规性校验、错误标记、智能纠错建议、校验结果导出
    3.分析统计：全局编码统计、维度细分统计、基础图表可视化展示
    4.基础能力：权限校验、数据库字典加载、千行级批量数据处理
- 必做页面：
    1.编码生成页面
    2.编码校验和修改页面
    3.分析统计页面
- 必做接口：
    1.11 项筛选条件字典数据加载接口
    2.编码实时生成接口
    3.编码批量校验接口
    4.临时保存/批量汇总接口
    5.Excel导入/导出接口
    6.编码生成记录入库/历史查询接口
    7.统计数据查询接口
    8.Auth 权限校验接口

## 非本期范围

- 暂不实现：
- 仅演示不落地：

## 边界说明

- 依赖外部系统的部分：
    1.必须接入公司Auth 权限服务完成登录与鉴权
    2.必须对接公司专属数据库获取编码字典与存储业务数据
    3.必须基于公司Base 基础服务规范开发接口与项目结构
- 可使用 Mock 的部分：
    1.开发阶段：编码字典表、权限校验逻辑可使用 Mock 数据
    2.联调前：Base 基础服务通用组件可临时 Mock 实现
    3.测试阶段：统计数据可 Mock 生成，验证图表展示效果
