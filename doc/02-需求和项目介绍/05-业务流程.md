# 业务流程

## 主流程

### 1. 编码生成流程

进入"编码生成"页面 → 左侧筛选条件区逐次选择类型、场站、二级类码等字典条件（支持从快速检索和手动新增字典补充条件）→ 点击"生成编码"按钮 → 右侧编码预览区显示生成的编码 → 可点击"保存至临时区"将编码保存至草稿列表 → 草稿区编码支持复制或导出 Excel → 点击"入库保存"将编码持久化至数据库 → 历史记录区可查看、删除、导出已入库的编码 → 筛选条件可自动保存为"最近条件"方便复用

### 2. 编码校验流程

进入"编码校验"页面 → 切换至"编码校验"标签页 → 选择输入方式（批量粘贴/上传 Excel 文件）→ 提交编码后点击"开始校验" → 系统按字典规则自动解析并分组展示 → 校验结果表格展示各段位识别情况，对于未识别的数据码，可点击"新增"补充字典项 → 识别完的数据带"已处理"或"已识别"标记

### 3. 编码解析流程

进入"编码校验"页面 → 切换至"编码解析"标签页 → 输入 31 位编码 → 点击"解析"按钮 → 系统自动将编码拆解为 11 个段位，分块卡片展示各段位的编码值和字典含义 → 下方编码结构图以彩色色块横向展示完整编码 → 未识别的段位标红提示

### 4. 重复编码稽核流程

进入"编码校验"页面 → 切换至"重复编码稽核"标签页 → 粘贴编码列表（每行一条）→ 点击"开始稽核"按钮 → 系统将每条编码与全量测点数据比对 → 稽核结果表格逐条展示"重复"（红色）或"不重复"（绿色）状态及说明

### 5. 编码修正流程

进入"编码校验"页面 → 切换至"编码修正"标签页 → 上传含"测点编码、测点描述、修改内容"三列的 Excel 文件 → 系统解析待修正编码列表 → 点击"开始修正" → 系统按修改内容解析需修改的段位，批量生成修正后的编码，自动检测是否与全量测点重复 → 修正过程显示进度条（超过 50 条自动分批处理）→ 修正结果表格展示修正前后编码逐段对比（旧值红色删除线，新值绿色加粗）、重复校验结果 → 点击"导出 Excel"保存修正结果

### 6. 字典查询流程

进入"字典管理"页面 → 默认展示"字典查询"标签页 → 分层树形结构展示全部字典项 → 展开/折叠各字典类型查看条目（类型域→二级类码→数据类码，数据码为懒加载）→ 字典项展示来源标签（集团统一/手动添加）

### 7. 新增字典记录查询流程

进入"字典管理"页面 → 切换至"字典新增"标签页 → 展示手动新增字典记录的分页列表（含类型、二级类码、数据类码、数据码、增加时间）→ 支持按类型和二级类码筛选 → 点击"导出 Excel"导出筛选后的记录

### 8. 场站管理流程

进入"字典管理"页面 → 切换至"场站维护"标签页 → 可查看场站分页列表 → 点击"新增"按钮单个添加场站 → 点击"批量新增"批量导入场站 → 点击编辑图标修改场站信息 → 点击删除图标移除场站 → 点击"全部删除"清空所有场站（需二次确认）→ 点击"导出"按钮导出场站列表为 Excel

### 9. 统计分析流程（编码生成统计）

进入"统计分析"页面 → 默认展示"编码生成统计"标签页 → 顶部概览卡片显示累计/今日/本周/本月生成数 → 类型维度的 4 色卡片展示风电/光伏/水电/不区分类型的编码数量 → 二级类码维度横向柱状图（可通过单选按钮切换风电/光伏/水电）→ 场站维度横向柱状图 → 下方分组统计列表展示各生成批次的编码详情，可展开查看单条编码，支持多选导出和批量删除

### 10. 统计分析流程（全量测点统计）

进入"统计分析"页面 → 切换至"全量测点统计"标签页 → 点击上传区域选择 .xlsx/.xls 文件 → 系统自动解析并导入（进度条显示导入进度和预估剩余时间）→ 导入过程中可点击"终止"按钮取消 → 导入完成后显示概览卡片（总测点数、风电/光伏/其他）→ 二级类码维度横向柱状图（支持类型切换）→ 场站维度横向柱状图 → 点击"清空数据"一键删除所有导入的测点数据

### 11. 数据源管理流程

进入"系统设置"页面 → 默认展示"数据源管理"标签页 → 表单默认锁定（不可编辑）→ 点击解锁按钮解锁 → 修改 PostgreSQL 连接参数（主机、端口、库名、Schema、用户名、密码、SSL 开关）→ 点击"测试连接"验证配置是否可用 → 连接成功后点击"保存配置" → 系统运行时动态切换数据库连接池，无需重启服务 → 保存成功后表单重新锁定

## 关键分支流程

- 分支1（编码生成多批次汇总）：单批次生成并临时保存 → 重复多轮"选条件→生成→暂存" → 草稿列表批量编辑/复制/导出 → 最终统一入库或导出
- 分支2（编码生成批量组合）：选择多个条件值（如多个二级类码）→ 系统自动进行笛卡尔积组合 → 批量生成多条编码 → 保存/导出
- 分支3（编码校验部分修正）：完成全量校验 → 筛选异常编码 → 查看纠错建议 → 手动修改或采纳系统建议 → 逐条确认修正 → 导出修改结果
- 分支4（快速复用最近条件）：进入生成页面 → 点击"最近条件" → 选择之前保存的筛选条件组合 → 系统自动填充各筛选条件 → 直接点击生成
- 分支5（编码修正分批处理）：修正数据超过 50 条 → 系统自动按 50 条一批分批调用后端 → 每批完成后更新进度条 → 全部批次完成后合并展示结果

## 异常流程

异常场景 1：筛选条件加载失败（数据库/接口异常）
处理方式：前端弹出明确提示，禁止生成操作，提供刷新重试按钮，日志记录异常信息

异常场景 2：Excel 导入格式错误/数据超限
处理方式：终止导入，提示"文件格式不合法或数据量超出限制"，引导使用标准模板、控制数据行数后重新上传

异常场景 3：编码生成无结果（条件冲突/未选全）
处理方式：提示"未生成有效编码，请检查筛选条件是否完整合规"，清空无效展示项

异常场景 4：导出/复制功能失败（浏览器限制）
处理方式：提示"操作失败，请检查权限或重试"，保留当前编码列表，支持重新发起操作

异常场景 5：全量测点导入中断
处理方式：导入过程中发生网络异常或服务中断，进度条停止，提示用户检查文件后重新上传；用户也可手动点击"终止"按钮取消正在进行的导入

异常场景 6：数据源连接测试失败
处理方式：表单中红色提示连接失败及具体原因（超时/拒绝连接/认证失败等），保持原有配置不变，不进行切换

异常场景 7：编码修正内容格式无法解析
处理方式：提示"修改内容格式无法识别，请参考模板格式重新填写"，展示正确的格式示例，引导重新上传

异常场景 8：重复编码稽核超限
处理方式：单次稽核超过 1000 条时，提示"单次稽核数量超出限制（上限 1000 条）"，请用户分批处理
