# 业务流程

## 主流程

### 0. 用户登录流程

进入登录页（`/login`）→ 输入用户名和密码 → 点击"登录"按钮 → 系统验证凭据 → 验证成功则签发 JWT Token 并跳转至首页（编码创建页面）→ 验证失败则提示"用户名或密码错误"→ 登录后所有请求自动携带 Token → Token 过期（24 小时）后自动跳转回登录页 → 根据用户所属租户（集团/云南省/福建省）加载对应区域的数据

### 1. 编码生成流程

进入"编码创建"页面 → 左侧筛选条件区逐次选择类型、场站、二级类码等字典条件（支持从快速检索搜索数据码和新增数据码补充条件）→ 点击"生成编码"按钮 → 右侧编码预览区显示生成的编码 → 可点击"保存至临时区"将编码保存至草稿列表 → 草稿区编码支持复制或导出 Excel → 点击"入库保存"将编码持久化至数据库 → 历史记录区可查看、删除、导出已入库的编码 → 筛选条件可自动保存为"最近条件"方便复用

### 2. 自动编码匹配流程

进入"编码创建"页面 → 点击"批量导入自动编码"按钮 → 打开向导对话框 → 步骤1：上传 Excel 文件（含设备名称、场站、二级类码等字段）→ 步骤2：将 Excel 列映射到系统字段（场站名称、二级类码、三级类码、数据类码、数据码、测点名称等）→ 步骤3：配置项目期号、并网线路、前缀号等固定参数 → 步骤4：系统自动匹配字典并批量生成编码 → 查看匹配结果，可导出或入库

### 3. 编码解析流程

进入"编码维护"页面 → 默认展示"编码解析"标签页 → 支持两种解析模式：
- **单条解析**：输入 31 位编码 → 点击"解析"按钮 → 系统自动将编码拆解为 11 个段位，分块卡片展示各段位的编码值和字典含义 → 下方编码结构图以彩色色块横向展示完整编码 → 未识别的段位标红提示
- **批量解析**：切换至"批量解析"子标签 → 粘贴多条编码（每行一条，最多 1000 条）→ 点击"开始解析"→ 表格展示每条编码的状态和错误信息 → 可展开查看详细段位 → 支持一键展开/收起所有行 → 点击"导出 Excel"保存结果

### 4. 重复编码稽核流程

进入"编码维护"页面 → 切换至"重复编码校验"标签页 → 粘贴编码列表（每行一条）→ 点击"开始稽核"按钮 → 系统将每条编码与全量测点数据比对 → 稽核结果表格逐条展示"重复"（红色）或"不重复"（绿色）状态及说明

### 5. 编码修正流程

进入"编码维护"页面 → 切换至"编码修正"标签页 → 上传含"测点编码、测点描述、修改内容"三列的 Excel 文件 → 系统解析待修正编码列表 → 点击"开始修正" → 系统按修改内容解析需修改的段位，批量生成修正后的编码，自动检测是否与全量测点重复 → 修正过程显示进度条（超过 50 条自动分批处理）→ 修正结果表格展示修正前后编码逐段对比（旧值红色删除线，新值绿色加粗）、重复校验结果 → 点击"导出 Excel"保存修正结果

### 6. 字典查询流程

进入"字典管理"页面 → 默认展示"编码字典查询"标签页 → 顶部显示字典概览卡片（风电/光伏/水电各维度的数据量统计）→ 支持两种查询模式：
- **数据码查询**：输入关键字模糊搜索 → 表格展示匹配的字典项
- **树形浏览**：分层树形结构展示全部字典项 → 展开/折叠各字典类型查看条目（类型域→二级类码→数据类码，数据码为懒加载）→ 字典项展示来源标签（集团统一/手动添加）

### 7. 新增字典记录查询流程

进入"字典管理"页面 → 切换至"字典新增"标签页 → 展示手动新增字典记录的分页列表（含类型、二级类码、数据类码、数据码、增加时间）→ 支持按类型和二级类码筛选 → 点击"导出 Excel"导出筛选后的记录

### 8. 字典审批流程

区域用户提交申请 → 进入"字典管理"页面的"字典新增"标签页 → 填写类型码、二级类码、数据类码、数据码、名称等信息 → 提交审批申请 → 集团管理员在系统配置中查看待审批列表 → 审核申请内容 → 选择"通过"自动补充字典数据 → 或选择"拒绝"并填写原因退回给区域用户 → 区域用户可查看自己的提交记录及审批状态

### 9. 场站管理流程

进入"系统配置"页面 → 切换至"场站管理"标签页 → 可查看场站分页列表 → 点击"新增"按钮单个添加场站 → 点击"批量新增"批量导入场站 → 点击编辑图标修改场站信息 → 点击删除图标移除场站 → 点击"全部删除"清空所有场站（需二次确认）→ 点击"导出"按钮导出场站列表为 Excel
（也可在"字典管理"页面的"场站维护"标签页进行相同操作）

### 10. 统计分析流程（编码生成统计）

进入"编码看板"页面 → 默认展示"编码生成统计"标签页 → 顶部概览卡片显示累计/今日/本周/本月生成数 → 类型维度的 4 色卡片展示风电/光伏/水电/不区分类型的编码数量 → 二级类码维度横向柱状图（可通过单选按钮切换风电/光伏/水电）→ 场站维度横向柱状图 → 下方分组统计列表展示各生成批次的编码详情，可展开查看单条编码，支持多选导出和批量删除

### 11. 统计分析流程（全量测点统计）

进入"编码看板"页面 → 切换至"编码测点分析"标签页 → 点击上传区域选择 .xlsx/.xls 文件 → 系统自动解析并导入（进度条显示导入进度和预估剩余时间）→ 导入过程中可点击"终止"按钮取消 → 导入完成后显示概览卡片（总测点数、风电/光伏/其他）→ 二级类码维度横向柱状图（支持类型切换）→ 场站维度横向柱状图 → 点击"清空数据"一键删除所有导入的测点数据

### 12. 时序稽核规则流程

进入"时序规则"页面 → 流水线式分步操作：
1. **导入场站数据**：上传场站 Excel 文件 → 系统解析并导入 → 实时显示导入进度
2. **导入测点数据**：上传测点 Excel 文件 → 系统解析并导入 → 实时显示导入进度
3. **生成规则**：点击"生成规则"按钮 → 系统自动生成四类稽核规则（死值/突变/越限/终端）→ 实时显示生成进度和各步骤完成情况
4. **导出结果**：
   - 按类型分别导出为单独 Excel 文件（sz/tb/yx/zd）
   - 全部类型打包为 ZIP 下载
   - 分批拆分导出（按 Z7 逻辑，适合超大数据量）
   - 支持总体结果单文件或分批 ZIP

### 13. 用户管理流程

进入"系统配置"页面 → 切换至"用户管理"标签页（仅集团管理员可见）→ 查看用户分页列表（含用户名称、区域、账号、最近登录时间）→ 点击"新增用户"按钮 → 填写用户名、显示名称、区域、密码 → 保存新增用户 → 点击编辑图标可修改用户信息 → 点击删除图标移除用户 → 系统预置 admin（集团管理员）/ yunnan（云南区域）/ fujian（福建区域）三个默认账号

### 14. 数据源管理流程

进入"系统配置"页面 → 默认展示"数据源管理"标签页 → 表单默认锁定（不可编辑）→ 点击解锁按钮解锁 → 修改 PostgreSQL 连接参数（主机、端口、库名、Schema、用户名、密码、SSL 开关）→ 点击"测试连接"验证配置是否可用 → 连接成功后点击"保存配置" → 系统运行时动态切换数据库连接池，无需重启服务 → 保存成功后表单重新锁定

### 15. 测点导入流程

进入"系统配置"页面 → 切换至"测点导入"标签页 → 拖拽或点击上传测点 Excel 文件 → 系统自动解析并导入（显示进度条和预估剩余时间）→ 导入过程中可点击"终止"按钮取消 → 导入完成后数据可在编码看板的测点分析中使用

## 关键分支流程

- 分支1（编码生成多批次汇总）：单批次生成并临时保存 → 重复多轮"选条件→生成→暂存" → 草稿列表批量编辑/复制/导出 → 最终统一入库或导出
- 分支2（编码生成批量组合）：选择多个条件值（如多个二级类码）→ 系统自动进行笛卡尔积组合 → 批量生成多条编码 → 保存/导出
- 分支3（快速复用最近条件）：进入生成页面 → 点击"最近条件" → 选择之前保存的筛选条件组合 → 系统自动填充各筛选条件 → 直接点击生成
- 分支4（编码修正分批处理）：修正数据超过 50 条 → 系统自动按 50 条一批分批调用后端 → 每批完成后更新进度条 → 全部批次完成后合并展示结果
- 分支5（自动编码匹配失败处理）：部分数据无法匹配字典 → 系统标记未匹配行并提示原因 → 用户调整数据后重新导入或手动补充字典

## 异常流程

异常场景 1：筛选条件加载失败（数据库/接口异常）
处理方式：前端弹出明确提示，禁止生成操作，提供刷新重试按钮，日志记录异常信息

异常场景 2：Excel 导入格式错误/数据超限
处理方式：终止导入，提示"文件格式不合法或数据量超出限制"，引导使用标准模板、控制数据行数后重新上传

异常场景 3：编码生成无结果（条件冲突/未选全）
处理方式：提示"未生成有效编码，请检查筛选条件是否完整合规"，清空无效展示项

异常场景 4：导出/复制功能失败（浏览器限制）
处理方式：提示"操作失败，请检查权限或重试"，保留当前编码列表，支持重新发起操作

异常场景 5：全量测点导入中断
处理方式：导入过程中发生网络异常或服务中断，进度条停止，提示用户检查文件后重新上传；用户也可手动点击"终止"按钮取消正在进行的导入

异常场景 6：数据源连接测试失败
处理方式：表单中红色提示连接失败及具体原因（超时/拒绝连接/认证失败等），保持原有配置不变，不进行切换

异常场景 7：编码修正内容格式无法解析
处理方式：提示"修改内容格式无法识别，请参考模板格式重新填写"，展示正确的格式示例，引导重新上传

异常场景 8：重复编码稽核超限
处理方式：单次稽核超过 10000 条时，提示"单次稽核数量超出限制（上限 10000 条）"，请用户分批处理

异常场景 9：用户登录认证失败
处理方式：登录页显示"用户名或密码错误"提示，不清空输入，供用户重新尝试

异常场景 10：Token 过期或无效
处理方式：前端路由守卫检测到未登录或 Token 无效时，自动跳转至登录页，并清除本地存储的认证信息
