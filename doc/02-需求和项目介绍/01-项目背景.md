# 项目背景

## 背景说明

- 本项目用于：中国华电新能源发电设备测点编码辅助作业，覆盖编码生成、合规校验、编码解析、编码修正、重复编码稽核、自动编码匹配、智能统计、字典管理、场站维护、字典审批、数据源管理、用户认证与权限管理、时序稽核规则生成等功能，为设备标识、数据采集、运维管理提供标准化编码支撑。
- 当前业务背景：测点编码是华电新能源设备管理的核心基础数据，编码规则包含 11 类维度字典项，体系复杂；当前完全依赖人工手动编制、核对，无统一工具支撑。同时缺少时序数据质量稽核规则自动生成工具，各区域独自维护字典数据缺少统一审批流程。
- 期望解决的问题：
  - 解决人工编码易出错、错误排查耗时长、批量校验效率低、标准不统一、数据无统计追溯等问题
  - 解决存量设备数据向时序稽核规则自动转换的问题
  - 解决区域字典数据新增缺乏统一审批管控的问题
  - 解决多区域用户访问权限与数据隔离的问题
  - 降低编码错误率、提升作业效率，推动测点编码标准化落地

## 培训价值

- 学员可通过本项目学习：
  - 电力新能源行业设备测点编码业务规则与企业级数据标准
  - Node.js/Express（TypeScript）+ Vue 3（Element Plus）前后端分离技术栈的实战开发
  - 数据库字典联动、批量数据处理、Excel 导入导出、数据可视化（ECharts）核心技能
  - 动态数据源切换、多租户架构、内存缓存、软删除等企业级开发模式
  - JWT 用户认证、权限控制、审批流程等通用业务模式
  - 轻量化业务工具从需求到验收的全流程设计与实现方法
- 本项目可演示的关键能力：
  - 多维度字典自动加载 + 编码实时生成的智能辅助能力
  - 单条 / 批量 / Excel 导入的编码解析与段位级对比能力
  - Excel 导入的批量编码修正与段位级新旧对比能力
  - 重复编码稽核：批量检测编码是否在存量测点中已存在的能力
  - 31 位编码智能解析与段位可视化展示能力
  - Excel 导入自动字典匹配与编码批量生成能力
  - 编码数据统计、维度细分与图表可视化能力
  - 按区域提交字典新增申请 + 集团审批的协同流程能力
  - 临时保存（草稿）、批量导出、数据入库的全流程作业能力
  - 运行时动态切换 PostgreSQL 数据源与多租户数据隔离能力
  - 时序稽核规则自动生成与多格式批量导出能力
  - 用户认证登录与多角色权限管理能力
  - 基于多租户 Schema 隔离的集团/区域数据隔离能力
  - 流水线式操作步骤依赖控制、页面刷新自动恢复进度能力
