# 用户与场景

## 目标用户

- 用户类型：
  1. 华电新能源测点编码专职工作人员
  2. 新能源场站设备管理辅助人员
  3. 各区域数据标准与运维管控人员（云南区域、福建区域等）
  4. 集团数据标准管控与审批人员（集团管理员）
  5. 时序数据质量稽核运维人员
- 用户特征：
  1. 熟悉新能源发电设备业务，掌握基础编码规则，无专业技术开发背景
  2. 日常高频处理单条/批量编码编制、核对、修改、上报与整改
  3. 对编码准确性、合规性、可追溯性要求严格
  4. 习惯使用公司内部统一系统，偏好简洁、少步骤、易上手的操作
  5. 需定期输出编码统计报表，用于工作汇报与管理复盘
  6. 集团管理员需管控多区域用户权限与字典数据变更

## 核心使用场景

- 场景1：系统登录与访问
  用户在登录页输入用户名和密码，系统验证身份后签发 JWT Token，根据用户所属区域（集团/云南省/福建省）加载对应租户的数据。不同区域的用户只能操作本区域的数据，集团管理员可管理所有区域。

- 场景2：新建设备批量编码生成
  编码人员按字典条件筛选，实时生成合规编码（单条或批量组合），可多次临时保存草稿后汇总，或直接入库保存。支持基于最近筛选条件快速复用生成，可在生成页面内直接通过快速检索面板搜索数据码并补充字典条件。最终导出 Excel 或入库留存，用于新建场站/设备的测点编码编制。

- 场景3：Excel 数据批量自动编码
  工作人员上传含设备名称、场站、二级类码等字段的 Excel 文件，通过列映射向导配置各列对应的系统字段，配置项目期号和前缀号等固定参数，系统自动匹配字典编码并批量生成 31 位测点编码。

- 场景4：编码解析与段位展示
  工作人员输入 31 位编码（单条或批量粘贴），系统自动解析各段位对应的字典含义，以分块卡片和编码结构图展示，支持辅助编码学习、排错与培训。批量解析结果支持展开查看详细段位和导出 Excel。

- 场景5：批量编码修正
  工作人员上传含"测点编码、测点描述、修改内容"的 Excel 文件，系统按修改内容解析出需要修改的段位（如二级类码、数据类码、数据码等），批量生成新的修正编码，自动检测修正后编码是否已在全量测点中重复，展示修改前后分段对比，最终导出修正结果。

- 场景6：重复编码稽核
  工作人员粘贴一批待新增的编码列表，系统自动与数据库全量测点数据进行比对，逐条标记"重复"或"不重复"，确保新增编码的唯一性。

- 场景7：字典查询与手动新增
  工作人员通过数据码查询或树形浏览两种模式查看字典项，使用关键字搜索快速定位特定字典条目，对缺失字典项可手动新增。支持查看手动新增字典记录并导出。

- 场景8：字典新增审批流程
  区域编码人员发现缺失的数据码/数据类码时，提交新增申请（含类型、二级类码、数据类码、数据码、名称等信息）。集团管理员查看待审批列表，审核通过后字典数据自动补充，或填写原因驳回申请。

- 场景9：场站信息管理
  工作人员维护新能源场站基础数据，支持单条新增、批量导入、编辑修改和删除，支持全部清空和 Excel 导出备份。

- 场景10：编码生成统计与管理汇报
  管控人员查看编码生成概览（累计/今日/本周/本月生成量），按类型、二级类码、场站等维度查看图表分布，查看分组详情列表，用于编码质量监控、标准落地复盘与工作汇报。

- 场景11：全量测点数据导入与分析
  管控人员上传存量测点编码 Excel 文件，系统自动解析导入（含进度提示与取消能力），导入完成后按类型、二级类码、场站维度进行统计分析和图表可视化展示。

- 场景12：时序稽核规则生成与导出
  运维人员按流水线步骤操作：先导入场站数据 Excel → 再导入测点数据 Excel → 系统自动生成死值/突变/越限/终端四类稽核规则 → 实时查看生成进度 → 完成后按类型导出 Excel 或打包下载全部 ZIP 文件，支持分批拆分导出。

- 场景13：用户账号管理
  集团管理员在系统配置中管理所有用户账号，可新增用户（分配用户名、显示名称、区域、密码）、编辑用户信息、删除用户，各区域用户只能使用本区域的账号登录。

- 场景14：动态切换数据源
  运维人员在系统配置中配置新的 PostgreSQL 数据源连接信息，测试连接成功后保存并生效，系统运行时动态切换数据库，无需重启服务。

- 场景15：测点数据导入
  工作人员在系统配置的测点导入页面上传测点数据文件，支持进度跟踪和取消操作，导入后数据可在看板和其他功能中使用。

## 用户关键诉求
  1. 操作极简：界面清晰、步骤少，无需培训即可快速使用
  2. 生成零错：严格按公司字典规则生成，彻底避免人工编码错误
  3. 批量高效：支持千行级导入导出、批量校验与批量修正，大幅提升作业效率
  4. 智能纠错：异常编码自动提示并给出修正方案，减少排查耗时
  5. 数据可溯：生成记录可保存、可查询，满足管理与审计要求
  6. 统计直观：图表可视化呈现合规情况，快速支撑管理决策
  7. 字典透明：可随时查阅全部字典项，支持区域提交审批补充
  8. 编码唯一：新增编码前可批量稽核是否与存量测点重复
  9. 配置灵活：数据源可动态切换，适应多环境部署需要
  10. 权限可控：用户认证登录，多区域数据隔离，集团统一管理账号
  11. 规则自动：时序稽核规则自动生成，支持多格式导出
