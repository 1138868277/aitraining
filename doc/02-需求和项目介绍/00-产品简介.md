# 华电新能源测点编码管理平台

**New Energy Measurement Point Code Management Platform**

---

## 平台定位

华电新能源测点编码管理平台是一站式、智能化的测点编码作业与管理工具，面向中国华电新能源发电业务，覆盖**编码生成、合规校验、字典管理、统计分析、时序规则**全业务流程。平台以标准化编码规则为基石，以智能化辅助手段为核心，从根本上解决人工编码易出错、校验效率低、标准不统一、数据难追溯等业务痛点，为新能源场站的设备标识、数据采集与运维管理提供坚实的编码基础设施。

---

## 核心价值

### 从人工到智能，重塑编码作业模式

测点编码是新能源设备管理的"数字身份证"，编码规则涵盖 11 类维度字典项，体系复杂、专业性强。传统模式下，编码员需手动查阅多张字典表逐段拼接编码，稍有不慎便产生错误，且错误排查犹如大海捞针。

本平台将编码规则内置于智能引擎之中，操作人员仅需点选条件，系统即可实时生成合规编码，彻底消除人工编制环节的出错可能，让编码作业从"手工作坊"迈入"智能工厂"时代。

### 从单点到全局，构建全流程闭环

平台打破了过去编码生成、校验、统计彼此割裂的局面，构建了"生成 → 校验 → 修正 → 分析 → 稽核"的完整业务闭环：

- **生成**：智能引擎驱动，单条与批量灵活适配，支持 Excel 导入自动匹配编码
- **校验**：31 位编码智能解析，批量解析校验与段位级对比
- **修正**：Excel 导入修正数据，批量执行编码段位修改，新旧编码对比展示
- **分析**：多维度可视化统计，支撑管理决策
- **稽核**：批量检测编码是否在存量测点中重复
- **字典**：全量数据透明可查，支持区域提交审批补充
- **规则**：自动生成时序数据稽核质量规则，支持多类型导出

### 从固定到灵活，适配多环境部署

平台支持运行时动态切换 PostgreSQL 数据源，支持多租户（集团/区域）数据隔离，无需重启服务即可完成数据库迁移或环境切换，极大提升了部署灵活性与运维效率。

### 从无证到可控，建立安全访问体系

平台内置用户认证与权限管理体系，支持多区域用户登录，集团管理员可统一管理用户账号与权限分配，确保系统操作可追溯、可管控。

## 核心功能

### 智能编码生成引擎

基于 11 类字典维度（设备类型、场站、项目期号、前缀号、一级类码、二级类码、二级类扩展码、三级类码、三级类扩展码、数据类码、数据码）的编码规则引擎，支持：

- **条件驱动生成**：操作人员通过下拉筛选条件逐层选择，系统自动拼接生成 31 位标准编码
- **批量组合生成**：支持多值条件自动笛卡尔积组合，一次性生成大批量合规编码
- **草稿暂存机制**：生成结果可暂存于草稿区，支持多轮生成后统一处理
- **最近条件复用**：自动记录常用筛选组合，一键填充历史条件
- **全生命周期管理**：从草稿、入库到历史查询与导出，完整追溯编码生命周期
- **字典快速检索**：在生成页面内嵌数据码模糊搜索，支持类型筛选与锁定

### 自动编码匹配（Excel 导入）

- **Excel 上传**：支持上传含名称、场站、二级类码等字段的 Excel 文件
- **列映射向导**：将 Excel 列映射到系统字段，灵活配置导入规则
- **自动字典匹配**：基于场站名称、二级类码、数据类码等自动匹配字典编码
- **编码配置**：支持配置项目期号、并网线路、前缀号等固定参数
- **批量限制**：单次最多处理 20000 条编码

### 多维编码解析与段位对比

提供编码的智能解析与段位级别对比能力：

- **单条解析**：输入 31 位编码，系统按 11 个段位拆解并反向解析字典含义
- **批量解析**：支持最多 1000 条编码的批量解析，表格展示并可展开查看详细段位
- **段位可视化**：分块卡片展示各段位编码与含义，编码结构图彩色分段展示
- **识别状态标记**：全部识别/部分未识别状态提示，未识别段位标红
- **结果导出**：批量解析结果支持导出为 Excel

### 重复编码稽核

批量检测编码是否已在全量测点中存在：

- **批量粘贴输入**：粘贴编码列表，一键稽核
- **逐条标记结果**：清晰标识"重复"或"不重复"状态
- **单次上限 10000 条**，快速高效

### 批量编码修正

支持对存量编码进行批量段位级修改：

- **Excel 模板导入**：上传含"测点编码、测点描述、修改内容"三列的 Excel 文件
- **分段对比展示**：修正前后编码按 11 个段位逐段对比，修改部分高亮标记（旧值标红删除线，新值标绿加粗）
- **重复自动检测**：修正后的编码自动与全量测点对比，标记重复风险
- **分批处理与进度跟踪**：超过 50 条自动分批处理，实时进度百分比显示
- **结果导出**：修正结果可导出为 Excel，便于线下备案
- **数量限制**：单次修正上限 100000 条

### 全量字典管理与场站维护

- **分层树形展示**：11 类字典数据以树形结构分层呈现，支持懒加载与逐层展开
- **两种查询模式**：数据码查询（关键字搜索）与树形浏览（层级导航）
- **字典项概览**：按风电/光伏/水电类型统计各字典维度数据量
- **手动新增字典**：支持单条和批量新增字典项，标记为手工添加
- **新增字典记录**：带筛选条件的手动新增字典记录查询，支持 Excel 导出
- **场站数据维护**：增删改查、批量新增、全部清空及 Excel 导出

### 字典审批流程

- **区域提交申请**：区域用户可提交缺少的数据码/数据类码新增申请
- **集团审批处理**：集团管理员可审核通过或驳回（需填写驳回原因）
- **全流程跟踪**：待审批/已审批列表分页展示，支持按区域和状态筛选

### 多维度统计分析看板

#### 编码生成统计
- **概览指标卡**：累计生成总数、今日/本周/本月生成量一目了然
- **类型分布**：风电、光伏、水电、不区分类型四维对比
- **二级类码分布**：按设备类型筛选的类码统计，横向柱状图直观对比
- **场站分布**：各场站编码生成量排行
- **分组详情列表**：按生成批次聚合展示，支持展开查看单条编码、多选导出与批量删除

#### 全量测点统计
- **大文件导入**：支持超大 Excel 文件上传解析，实时进度条与预估剩余时间
- **导入可取消**：导入过程支持随时终止，避免长时间等待
- **多维度分析**：总测点数、类型分布、二级类码分布、场站分布全维度覆盖
- **数据管理**：一键清空导入数据，灵活控制分析范围

### 时序稽核规则引擎

- **数据导入**：支持导入场站数据和测点数据（Excel 格式）
- **规则生成**：自动生成四类时序稽核规则——死值（sz）、突变（tb）、越限（yx）、终端（zd）
- **进度跟踪**：分步骤可视化流水线，实时展示各阶段处理进度
- **多方式导出**：
  - 按类型分别导出单个 Excel 文件
  - 全部类型打包为 ZIP 下载
  - 分批拆分导出（按 Z7 逻辑拆分）
  - 整体与拆分 ZIP 同时提供

### 运行时动态数据源

- **可视化配置**：在系统设置页面即可完成 PostgreSQL 数据源参数配置
- **连接验证**：测试连接功能确保配置正确无误后再保存
- **热切换生效**：保存即生效，无需重启服务，无缝切换数据库环境
- **锁定保护**：配置表单默认锁定，防止非授权修改

### 用户认证与权限管理

- **登录认证**：用户名+密码登录，JWT Token 鉴权，24 小时有效期
- **多租户架构**：集团/云南省/福建省三区域数据隔离，各区域操作各自数据
- **用户管理**：集团管理员可新增、编辑、删除用户，支持按区域分配账号

## 技术架构

- **后端**：Node.js + Express 框架，TypeScript 语言，模块化分层架构（Controller → Service → Domain）
- **前端**：Vue 3 组合式 API + Element Plus 组件库 + ECharts 数据可视化
- **数据库**：PostgreSQL，支持运行时动态切换与多租户数据隔离
- **工程体系**：pnpm 多包仓库，前后端分离，共享类型定义（`@cec/contracts`）

## 总结
