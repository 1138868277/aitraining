# 编码校验

## 功能目标

- 提供字典数据的分层树形查询与快速检索，方便用户查阅全部字典项
- 支持 31 位测点编码的智能解析，反向展示各段位对应的字典含义
- 支持批量粘贴和上传 Excel 两种导入方式，对编码进行合规校验与段位识别
- 对未识别字典项提供手动新增入口，补充缺失字典数据
- 提供场站基础数据维护功能

## 页面说明

### 页面路径

- `/code-validate` — 编码校验主页面

### 页面布局

页面采用单卡片多标签页布局，共 5 个标签页：

#### Tab1 — 字典查询

包含字典概览卡片和树形展示区：

**字典概览卡片（DictOverviewCard 组件）**
- 树形结构的字典概览：显示各层级的字典数据
- 懒加载模式，按需展开加载子节点

**树形展示区**
- 工具栏："收起所有"按钮
- 使用 el-tree 组件，懒加载加载子节点
- 树节点展示：类型标签 + 编码 + 名称 + 子项数量 + 来源标识（手动添加/集团统一）
- 支持空数据展示

#### Tab2 — 编码解析

**输入区**
- 31 位编码输入框（textarea，maxlength=31），实时显示已输入位数
- "解析"按钮（编码满 31 位且输入变化后启用）

**解析结果展示**
- 编码分段展示：以段位块的形式展示各段编码、对应的标签和名称
- 编码可视化：以不同颜色展示各段在完整编码中的分布
- 识别状态标签："全部识别"（绿色）或"部分未识别"（红色）
- 未识别段显示"未识别"标识和错误提示

#### Tab3 — 编码校验

**输入区域**
- 两种输入模式切换：
  - 批量粘贴：textarea 输入编码列表（每行一条：编码 名称），自动解析编码和名称
  - 上传 Excel：拖拽/点击上传 .xlsx/.xls 文件，自动读取"测点编码"和"测点描述"列
- 显示已解析编码数量
- 操作按钮："开始校验"、"清空"

**校验结果**
- 展开式表格（el-table），按类型/二级类码/数据类码/数据码分组展示
- 每行展示：类型、二级类码、数据类码、数据码的编码和名称
- 展开行展示该分组下的具体测点编码列表
- 未识别的段位显示"未识别"红色标记
- 操作列：对未识别但可新增的项显示"新增"按钮
- 已处理项标记"已处理"
- 操作区包含"下载模板"链接

#### Tab4 — 字典新增

**页面功能**
- 展示手动新增编码字典的记录列表
- 列表字段：类型、二级类码、数据类码、数据码、增加时间
- 支持类型筛选和二级类码筛选
- 支持"刷新"和"导出 Excel"操作
- 分页展示

#### Tab5 — 场站维护

嵌入 StationTab 组件，提供场站基础数据 CRUD 功能（详见场站管理设计）。

### 新增对话框

点击校验结果中的"新增"按钮弹出新增字典对话框：
- 展示当前二级类码信息（只读）
- 数据类码（只读，自动填充）
- 数据码（可编辑，3位数字）
- 数据码名称（可编辑）
- 确认新增后调用手动新增接口

## 输入输出

### 批量解析编码

**输入：**
```json
{
  "codes": [
    { "code": "string", "name": "string" }
  ]
}
```

**输出：**
```json
[
  {
    "typeCode": "F",
    "typeName": "风电",
    "secondClassCode": "001",
    "secondClassName": "风机",
    "dataCategoryCode": "01",
    "dataCategoryName": "温度",
    "dataCode": "001",
    "dataName": "环境温度",
    "matchedCodes": [
      { "code": "31位编码", "name": "编码名称" }
    ]
  }
]
```

### 编码解析（31位）

**输入：** `code`（31 位编码字符串）

**输出：**
```json
{
  "rawCode": "31位编码",
  "segments": [
    { "label": "场站", "code": "0001", "name": "某场站" },
    { "label": "类型", "code": "F", "name": "风电" }
  ],
  "isValid": true,
  "errorMessage": null
}
```

### 手动新增字典

**输入：**
```json
{
  "typeCode": "F",
  "secondClassCode": "001",
  "dataCategoryCode": "01",
  "dataCategoryName": "温度",
  "dataCode": "002",
  "dataName": "环境温度2"
}
```

## 业务规则

### 字典树规则

- 字典树分三层加载：类型域 → 二级类码 → 数据码
- 懒加载：初始只加载顶层类型域，展开时动态加载子节点
- 树节点显示编码、名称和子项数量

### 编码解析规则

- 输入必须为 31 位字母/数字组合
- 按预定义段位分割编码，逐个段位查询字典表
- 已识别段显示编码和名称，未识别段标记"未识别"
- 编码可视化用不同颜色区分各段位

### 校验规则

- 输入编码列表按类型/二级类码/数据类码/数据码分组展示
- 各段位逐一查询字典表，未命中标记"未识别"
- 自动去重：相同编码合并展示
- 识别完整（类型+二级类码+数据类码+数据码均已识别）的行标记"已识别"
- 仅数据码未识别的行可进行"新增"操作（补充字典后自动处理）

### 字典新增规则

- 新增字典项时填充默认数据码（系统自动推荐最大值+1）
- 新增的字典项标记 `is_manual = '1'`
- 字典新增后前端校验结果自动更新

### 场站管理规则

详见场站管理设计文档。

## 异常处理

| 异常场景 | 前端提示 |
|---------|---------|
| 字典树加载失败 | "字典数据加载中..." + 加载失败提示 |
| 编码解析输入不足 31 位 | "编码必须为31位"，解析按钮禁用 |
| 校验编码列表为空 | "请先输入待校验的编码" |
| 上传文件格式错误 | "文件格式不合法，请上传.xlsx或.xls文件" |

## 验收标准

1. 字典树懒加载正确，展开/收起正常
2. 31 位编码解析正确展示各段位含义
3. 编码可视化颜色分段展示清晰
4. 批量粘贴自动解析编码和名称
5. Excel 上传正确读取测点编码和测点描述列
6. 校验结果按分组展开展示，段位未识别正确标记
7. 未识别项"新增"按钮弹出对话框，填写保存后字典更新
8. 字典新增记录列表分页正确，支持筛选和导出
9. 场站维护标签页 CRUD 功能正常
