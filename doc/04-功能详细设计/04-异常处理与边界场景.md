## 通用异常

| 异常类型 | HTTP 状态码 | 错误码 | 提示信息 |
|---------|-----------|-------|---------|
| 参数缺失 | 400 | MISSING_PARAMETER | 请填写必填参数：{field} |
| 参数格式错误 | 400 | PARAM_FORMAT_ERROR | 参数{field}格式不正确 |
| 资源不存在 | 404 | RESOURCE_NOT_FOUND | 请求的资源不存在 |
| 重复提交 | 409 | DUPLICATE_SUBMISSION | 请勿重复提交 |
| 请求超时 | 504 | REQUEST_TIMEOUT | 请求超时，请重试 |
| 系统异常 | 500 | SYSTEM_ERROR | 系统繁忙，请稍后重试 |
| 外部服务异常 | 503 | EXTERNAL_SERVICE_ERROR | 外部服务暂不可用，请稍后重试 |

## 全局异常处理

所有未捕获的异常由 Express 全局错误处理中间件统一处理：

```typescript
// 全局错误处理中间件
function globalErrorHandler(err, req, res, next) {
  // 记录错误日志
  console.error('Unhandled error:', err);

  // 返回统一错误响应，不暴露内部错误详情
  res.status(500).json({
    code: 'SYSTEM_ERROR',
    message: '系统繁忙，请稍后重试',
    data: null,
    timestamp: Date.now(),
  });
}
```

## 边界场景

### 数据边界

| 场景 | 处理方式 |
|-----|---------|
| 空数据 | 返回 code=0，data 为空数组或 null，前端展示空状态提示 |
| 超大数据量 | 超出模块限制时提示分批处理，前端控制上传/输入上限 |
| 特殊字符 | 编码中只允许字母和数字，输入时前端自动过滤非法字符 |

### 操作边界

| 场景 | 处理方式 |
|-----|---------|
| 条件未选全点击生成 | 生成按钮置灰，不可点击 |
| 编码列表为空点击校验 | 校验按钮置灰，不可点击 |
| 无统计数据 | 图表区域显示空状态引导提示 |
| 服务重启 | 草稿数据和校验任务数据丢失（存储于内存），提示用户重新操作 |
| 网络断开后重连 | 保留当前页面状态，手动刷新后可继续操作 |

### 输入校验边界

| 字段 | 校验规则 |
|-----|---------|
| 编码格式 | 只能包含字母和数字，固定 31 位 |
| 编码列表 | 每行一条，去除首尾空格和空行 |
| 上传文件 | 仅支持 .xlsx/.xls |

## 处理原则

- **统一异常处理**：Express 全局错误处理中间件统一捕获异常，返回标准 `{ code, message, data, timestamp }` 格式
- **用户友好提示**：业务异常提供清晰的中文提示，系统异常提供通用提示，不暴露内部细节
- **日志记录**：记录请求方法、路径、响应状态码和耗时（通过 `requestLogger` 中间件）
- **超时控制**：常规请求 30 秒超时，全量测点导入 600 秒超时（大文件场景）
- **前端防重复**：提交后按钮置灰并显示加载状态，防止重复点击
