# 字典管理

## 功能目标

- 提供字典数据的分层树形查询与快速检索，方便用户查阅全部字典项
- 支持字典概览统计卡片，按风电/光伏/水电类型展示各维度字典数据量
- 支持数据码查询和树形浏览两种模式切换
- 支持查看和筛选手动新增字典记录，支持 Excel 导出
- 提供场站基础数据维护功能（增删改查、批量操作、导出）

## 页面说明

### 页面路径

- `/code-validate` — 字典管理主页面

### 页面布局

页面采用单卡片多标签页布局，共 3 个标签页。

#### Tab1 — 编码字典查询

**字典概览卡片（DictOverviewCard 组件）**
- 按风电/光伏/水电三种类型展示各维度字典项数量统计
- 展示维度：类型域、二级类码、三级类码、数据类码、数据码

**查询模式切换**
- 两种模式：数据码查询（el-input 关键字搜索）与树形浏览（层级导航）
- 默认显示树形浏览模式

**数据码查询模式**
- 关键字搜索输入框，支持模糊匹配
- 展示符合条件的数据码列表

**树形浏览模式**
- 工具栏："收起所有"按钮
- 使用 el-tree 组件，懒加载加载子节点（前三级从缓存加载，数据码层从 API 懒加载）
- 树节点展示：类型标签 + 编码 + 名称 + 子项数量 + 来源标识（"手动添加"黄色标签/"集团统一"灰色标签）
- 支持空数据展示（el-empty）

**树节点层级：**
- 第一层：类型域（typeDomain），如 F 风电、G 光伏、Y 水电
- 第二层：二级类码（secondClass），如 001 风机
- 第三层：数据类码（dataCategory），如 01 温度
- 第四层：数据码（dataCode），懒加载，如 001 环境温度

#### Tab2 — 字典新增

**筛选栏**
- 类型筛选（el-select，下拉选择）
- 二级类码筛选（el-select，下拉选择），切换类型时重置
- 选择变化后自动触发查询

**记录列表**
- 展开式表格展示手动新增编码字典记录
- 列表字段：序号、类型、二级类码、数据类码、数据码、增加时间
- 操作按钮："刷新"、"导出 Excel"
- 分页控制（每页 10/20/50/100 条）
- 支持空数据展示（el-empty）

#### Tab3 — 场站维护

嵌入 StationTab 组件，提供场站基础数据 CRUD 功能（详见下方"场站管理"设计）。

**场站列表**
- 分页展示场站信息（编号、场站名称、管理区域等）
- 操作列：编辑、删除

**操作按钮**
- "新增"：弹出对话框，填写场站信息后保存
- "批量新增"：弹出对话框，批量输入场站数据
- "全部删除"：二次确认后清空所有场站
- "导出"：导出场站列表为 Excel

## 接口说明

### 字典查询

| 接口名称 | 方法 | 路径 | 说明 |
|---------|------|------|------|
| 获取字典树 | GET | /api/dict/tree | 获取完整字典树（前三级缓存） |
| 获取数据类码树 | GET | /api/dict/tree/data-codes | 懒加载获取数据类码下的数据码 |
| 快速检索字典 | GET | /api/dict/quick-search | 关键字搜索字典项 |
| 字典概览统计 | GET | /api/dict/statistics | 按类型统计各维度字典数据量 |

### 字典新增记录

| 接口名称 | 方法 | 路径 | 说明 |
|---------|------|------|------|
| 字典手动统计 | GET | /api/dict/manual-statistics | 分页查询手动新增字典记录（含类型和二级类码筛选） |
| 导出手动统计 | GET | /api/dict/manual-statistics/export | 导出手动新增记录为 Excel |

### 场站管理

| 接口名称 | 方法 | 路径 | 说明 |
|---------|------|------|------|
| 获取场站列表 | GET | /api/station/list | 分页获取场站列表 |
| 新增场站 | POST | /api/station | 单个新增场站 |
| 批量新增场站 | POST | /api/station/batch | 批量导入场站 |
| 编辑场站 | PUT | /api/station/:id | 修改场站信息 |
| 删除场站 | DELETE | /api/station/:id | 删除指定场站 |
| 清空场站 | DELETE | /api/station/all | 删除所有场站 |
| 导出场站 | GET | /api/station/export | 导出场站列表为 Excel |

## 业务规则

### 字典树规则

- 字典树分四级：类型域 → 二级类码 → 数据类码 → 数据码
- 前三级从内存缓存加载，数据码层从 API 懒加载
- 树节点显示编码、名称、子项数量和来源标识（手动添加/集团统一）
- 字典概览卡片显示各类型字典数量统计

### 字典查询模式规则

- 树形浏览模式：按层级展开浏览全部字典项
- 数据码查询模式：输入关键字模糊搜索，快速定位字典项
- 两种模式可随时切换

### 字典新增记录规则

- 记录来源为 `is_manual = '1'` 的手动新增字典项
- 支持按类型和二级类码筛选
- 导出时筛选条件同样生效

### 场站管理规则

- 场站编码唯一，新增时自动检测重复
- 全部删除操作需二次确认
- 批量新增支持多条同时导入

## 异常处理

| 异常场景 | 前端提示 |
|---------|---------|
| 字典树加载失败 | "字典树加载失败" + 错误提示 |
| 字典概览统计加载失败 | "统计数据加载失败" + 错误提示 |
| 字典新增记录加载失败 | "加载统计数据失败" + 错误提示 |
| 导出失败 | "导出失败" + 错误提示 |
| 场站操作失败 | 对应操作的具体错误提示 |

## 验收标准

1. 字典概览卡片按类型展示各维度统计数据正确
2. 字典树懒加载正确，展开/收起正常
3. 树节点展示编码、名称、来源标签和子项数量
4. 数据码查询模式关键字搜索正确
5. 树形浏览与数据码查询两种模式切换正常
6. 字典新增记录列表分页正确，支持类型和二级类码筛选
7. 字典新增记录导出 Excel 内容正确
8. 场站维护标签页 CRUD 功能正常
9. 场站批量新增和全部清空功能正常
10. 场站导出 Excel 内容正确
