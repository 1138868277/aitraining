# 编码生成

## 功能目标

- 提供基于字典的筛选条件面板，支持快速检索字典项和手动补充缺失数据码
- 根据用户选定的条件组合，自动生成合规的 31 位测点编码及编码名称
- 支持单条生成和批量生成（扩展码范围组合、二级类码多选笛卡尔积）
- 支持草稿临时保存、最近筛选条件复用、编码历史记录管理
- 支持 Excel 上传，通过列映射向导自动匹配字典并批量生成编码（自动编码）
- 支持编码的复制、Excel 导出和数据库入库持久化

## 页面说明

### 页面路径

- `/code-generate` — 编码创建主页面

### 页面布局

页面从上到下依次为：顶部横幅 → 快速检索工具栏 → 编码生成面板（含手动/自动两个子标签页） → 编码结果面板（含预览/列表两个子标签页）。

#### 顶部横幅

- 标题"编码创建"，描述"按规则筛选条件批量生成测点编码，支持预览、导出与修正"

#### 快速检索工具栏

- 标题"快速检索" + "新增数据码"按钮（弹出手动新增字典对话框）
- 搜索行：输入框（支持数据码模糊搜索或编码后5位精确搜索）+ 最近搜索标签（带关闭按钮）
- 类型筛选行：F(风电)/G(光伏)/S(水电) 单选按钮组 + 锁定按钮（锁定后切换其他条件时类型值不变）
- 搜索结果表格：列包含类型域、二级类码、数据类码、数据码、来源（集团统一/手动添加）、操作（"筛选"按钮，点击自动填充生成条件）
- 支持分页控制

#### Tab1 — 手动编码

- 筛选条件表单，每行 4 个条件项，字段类型分为下拉选择、文本输入、扩展数输入三种：
  - 场站（stationCode）：下拉选择，单选，支持锁定
  - 发电类型（typeCode）：下拉选择，单选，支持锁定
  - 二级类码（secondClassCode）：下拉选择，支持多选，支持锁定
  - 三级类码（thirdClassCode）：下拉选择，单选
  - 数据类码（dataTypeCode）：下拉选择，单选
  - 数据码（dataCode）：下拉选择
  - 二级扩展码（secondExtCode）：扩展数字输入（起始值 + 延伸数量），默认从 1 开始
  - 三级扩展码（thirdExtCode）：扩展数字输入（起始值 + 延伸数量），默认从 0 开始
- 条件锁定功能：点击锁定按钮可固定某个条件值，切换其他条件时该值不变
- 快捷选项：部分字段提供快捷标签（如"不限"）
- 操作按钮：生成编码、清空条件
- "最近记录"按钮（弹出最近筛选条件列表，点击自动填充）
- 多选 + 延伸数量组合时显示预计生成编码数
- 单次生成上限 100 条

#### Tab2 — 自动编码（AutoCodePanel 组件）

三步向导式操作：

**Step 1 — 上传文件**
- 拖拽/点击上传 .xlsx/.xls 文件
- 支持下载导入模板
- 显示已选择的文件名

**Step 2 — 列映射配置**
- 系统自动检测 Excel 列名，与系统字段建立映射
- 配置字段：场站名称、二级类码、三级类码、数据类码、数据码（映射到字典编码）
- 映射方式：下拉选择 Excel 中的列名对应到系统字段
- 支持模糊匹配自动推荐映射

**Step 3 — 编码配置与生成**
- 配置固定参数：项目期号（projectLineCode，3 位）、前缀号（prefixNo，1 位）、一级类码（firstClassCode，2 位）
- 选择发电类型（typeCode）
- 匹配进度展示：显示已匹配/总数
- "开始匹配"按钮触发自动字典匹配与编码生成
- 单次最多处理 20000 条编码
- 匹配完成后跳转至编码预览展示结果

#### 编码结果（含两个子标签页）

- Tab1 — 编码预览：展示最近一次生成的编码列表，支持复制全部、保存入库、单条复制、行内编辑编码名称。两次生成之间自动叠加，不覆盖。
- Tab2 — 编码列表：展示已入库保存的编码历史记录，支持类型/场站/二级类码等筛选条件、分页、导出（Excel）、单条/批量删除

```json
  "secondClassCode": "string | string[]",
  "thirdClassCode": "string",
  "dataTypeCode": "string",
  "dataCode": "string",
  "secondExtCodeStart": "number",
  "secondExtCodeCount": "number",
  "thirdExtCodeStart": "number",
  "thirdExtCodeCount": "number"
}
```

**输出：**
```json
{
  "codes": [
    {
      "code": "ABCDEFGHIJKLMNOPQRSTU12345678901",
      "name": "编码名称",
      "generateTime": "2025-01-01 12:00:00"
    }
  ]
}
```

**输入：**
```json
{
  "rows": [
    { "stationName": "某场站", "secondClassName": "风机", "thirdClassName": "", "dataCategoryName": "温度", "dataName": "环境温度" }
  ],
  "config": {
    "typeCode": "F",
    "projectLineCode": "001",
    "prefixNo": "1",
    "firstClassCode": "01"
  }
}
```

**输出：**
```json
{
  "codes": [
    { "code": "31位编码", "name": "编码名称", "generateTime": "2025-01-01 12:00:00" }
  ],
  "totalCount": 100,
  "matchedCount": 98,
  "failedCount": 2

```

```json

}
```

## 业务规则

### 编码规则

- 测点编码由多段编码维度拼接而成，共 31 位
- 编码段位构成：场站编码(4位) + 类型(2位) + 项目线(3位) + 前缀(1位) + 一级类码(2位) + 二级类码(3位) + 二级扩展码(4位) + 三级类码(3位) + 三级扩展码(4位) + 数据分类码(2位) + 数据码(3位)
- 编码名称由各段字典项名称拼接生成

### 条件筛选规则

- 必选条件：至少场站、发电类型、二级类码必须选择
- 条件联动：选择上级后下级自动加载对应选项（如选类型后二级类码列表/数据类码联动）
- 功能锁定：条件可单独锁定，切换其他条件时锁定值不变，方便固定某些维度调整其他维度生成

### 批量生成规则

- 二级类码支持多选，多个值自动进行笛卡尔积组合生成多条编码
- 二级扩展码和三级扩展码支持设置起始值和延伸数量，自动生成连续编号范围内的编码
- 批量生成时显示预计生成编码数量
- 单次生成上限 100 条

### 自动编码规则

- Excel 上传后自动检测列名，引导用户完成列映射配置
- 系统根据场站名称、二级类码名称等在字典表中模糊匹配对应的编码值
- 固定参数（项目期号、前缀号、一级类码）由用户在向导中统一配置
- 单次处理上限 20000 条
- 匹配失败的行记录失败原因，不影响其他行生成

### 草稿（临时区）规则

- 草稿数据存储于后端内存（以 x-session-id 隔离），服务重启后丢失
- 草稿上限 5000 条
- 支持多轮生成后统一处理（入库或导出）

### 最近条件规则

- 每次生成编码后，当前的筛选条件组合自动保存为最近条件
- 最近条件列表在"最近记录"弹出面板中展示，显示各字段值和摘要
- 点击最近条件项自动填充筛选条件并触发生成

### 导出规则

- 导出文件格式：.xlsx
- 单次导出上限：10000 行

## 异常处理

| 异常场景 | 前端提示 |
|---------|---------|
| 字典数据加载失败 | "筛选条件加载失败，请刷新重试" |
| 必选条件未选全 | 生成按钮置灰不可点击 |
| 单次生成超限 | "单次生成数量超出限制（上限100条）" |
| 草稿区已满 | "临时区已满（上限5000条），请先导出或清理" |
| 自动编码文件格式错误 | "文件格式不合法，请上传.xlsx或.xls文件" |
| 自动编码超限 | "单次处理数量超出限制（上限20000条）" |
| 列映射未完成 | "请完成所有必需的列映射" |
| 字典匹配失败 | "部分数据字典匹配失败，请检查场站名称和类码名称是否正确" |

## 验收标准

1. 快速检索支持关键字搜索和类型筛选，结果正确
2. "新增数据码"按钮弹出对话框，填写保存后字典数据更新
3. 最近搜索标签点击自动填充搜索框，关闭后消失
4. 最近记录点击后正确填充筛选条件
5. 筛选条件联动正确，锁定功能生效
6. 生成编码为 31 位，段位构成正确
7. 批量生成（多选+扩展码）编码数量计算正确
8. 自动编码三步向导流程完整：上传 → 列映射 → 配置生成
9. 自动编码结果正确展示并支持保存入库
10. 编码预览列表正确展示，支持复制、保存、叠加
11. 编码列表（历史记录）筛选、分页、导出、删除正常
