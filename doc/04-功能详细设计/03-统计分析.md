# 统计分析

## 功能目标

- 提供编码生成数据的多维度统计分析，包括概览、类型分布、二级类码分布、场站分布、分组详情
- 支持全量测点数据的 Excel 导入、解析存储与多维度统计分析
- 提供直观的 ECharts 图表可视化展示（横向柱状图，渐变色带阴影）

## 页面说明

### 页面路径

- `/statistics` — 编码看板主页面

### 页面布局

页面采用左侧标签导航 + 右侧内容区布局，共 2 个标签页。

#### Tab1 — 编码生成统计（CodeGenTab 组件）

**概览卡片（4 个）**
| 卡片 | 说明 | 颜色主题 |
|------|------|---------|
| 累计生成 | 所有已入库编码的总数 | 蓝色 |
| 今日生成 | 当日生成的编码数 | 绿色 |
| 本周生成 | 本周生成的编码数 | 青色 |
| 本月生成 | 本月生成的编码数 | 橙色 |

**类型维度（4 个卡片）**
以带图标的统计卡片展示四种类型的编码计数：
- 风电（蓝色）
- 光伏（绿色）
- 水电（青色）
- 不区分类型（橙色）

**二级类码维度（ECharts 横向柱状图）**
- 顶部通过单选按钮切换风电/光伏/水电三种类型
- 横向柱状图展示选中类型下各二级类码的编码数量
- 渐变色柱状图，带阴影效果

**场站维度（ECharts 横向柱状图）**
- 横向柱状图展示各场站的编码数量
- 绿色渐变色柱状图

**编码生成列表（分组统计表格）**
- 筛选栏：类型、场站、二级类码、数据类码 + 查询/重置按钮
- 展开式表格，按生成批次/分组聚合展示
- 主行显示分组信息（类型、场站、二级类码等）
- 展开行显示该分组下的单条测点编码列表
- 复选框多选 + 批量操作：导出选中（XLSX）、删除选中
- 分页控制

#### Tab2 — 编码测点分析（MeasureTab 组件）

**导入区**
- "选择文件"按钮触发隐藏的 file input，支持 .xlsx/.xls
- 选中文件后显示文件名
- 导入中显示进度条（el-progress，带百分比和颜色变化）+ 预估剩余时间
- 导入中显示"终止"按钮，可取消正在进行的导入
- 导入完成后显示结果消息
- 已导入状态显示"清空数据"按钮（红色，带二次确认）
- 显示业务数据最新导入时间

**概览卡片（导入数据量 > 0 时显示）**
| 卡片 | 说明 |
|------|------|
| 总测点数 | 导入的测点数据总量 |
| 风电 | 类型域为风电的测点数 |
| 光伏 | 类型域为光伏的测点数 |
| 其他 | 其他类型的测点数 |

**图表区（导入数据量 > 0 时显示）**
- 二级类码维度：横向柱状图，单选按钮切换风电/光伏
- 场站维度：横向柱状图，高度根据场站数量自适应

**空状态**：未导入数据时显示"请先导入测点数据"空状态提示

## 接口说明

### 编码生成统计

| 接口名称 | 方法 | 路径 | 说明 |
|---------|------|------|------|
| 生成概览 | GET | /api/statistics/code-gen/overview | 累计/今日/本周/本月生成数 |
| 类型分布 | GET | /api/statistics/code-gen/by-type | 风电/光伏/水电/不区分类型统计 |
| 二级类码分布 | GET | /api/statistics/code-gen/by-second-class | 按二级类码聚合统计，参数 ?type=wind/solar/hydro |
| 场站分布 | GET | /api/statistics/code-gen/by-station | 按场站聚合统计 |
| 生成记录列表 | GET | /api/statistics/code-gen/list | 分页查询分组记录 |
| 分组详情 | GET | /api/statistics/code-gen/group-detail | 查询指定分组的单条编码 |
| 删除分组 | POST | /api/statistics/code-gen/delete-groups | 批量删除生成分组 |
| 生成趋势 | GET | /api/statistics/code-gen/trend | 按时间维度的趋势数据 |

| 接口名称 | 方法 | 路径 | 说明 |
|---------|------|------|------|

| 查询导入状态 | GET | /api/statistics/measurement/import-status | 查询导入任务进度 |
| 取消导入 | POST | /api/statistics/measurement/cancel-import | 取消正在进行的导入 |
| 测点概览 | GET | /api/statistics/measurement/overview | 总测点数/风电/光伏/其他 |
| 二级类码分布 | GET | /api/statistics/measurement/by-second-class | 按二级类码聚合统计，参数 ?type=wind/solar |
| 场站分布 | GET | /api/statistics/measurement/by-station | 按场站聚合统计 |

## 业务规则

### 编码生成统计规则

- 概览数据统计自 `cec_new_energy_createcode` 表
- 今日/本周/本月按 `create_tm` 时间字段过滤
- 类型维度按 `type_code` 字段统计分组
- 二级类码维度按设备类型（风电/光伏/水电）筛选后聚合
- 场站维度按 `station_code` 字段聚合

### 全量测点导入规则

- 支持 .xlsx/.xls 格式，文件大小上限 200MB
- 自动检测 Excel 中包含"测点编码"或"code"的列，提取 31 位编码
- 后端将 31 位编码按位段解析后分段存储至 `cec_new_energy_measurement_points` 表
- 每 500 条一批批量插入
- 导入过程支持取消（设置取消标志，下一批插入前终止）
- 导入完成后前端停止轮询

### 图表规则

- 二级类码图表：各分类名称显示在 Y 轴，数量为 X 轴横向柱状图
- 场站图表：场站名称显示在 Y 轴，编码/测点数量为 X 轴横向柱状图
- 柱状图使用渐变色，带阴影效果
- 鼠标悬浮显示具体数值

## 数据关联

| 表名 | 用途 |
|------|------|
| cec_new_energy_createcode | 编码生成统计数据源 |
| cec_new_energy_measurement_points | 全量测点统计数据源 |
| cec_new_energy_station_dict | 场站名称映射 |
| cec_new_energy_type_dict | 类型域名称映射 |
| cec_new_energy_second_class_type_dict | 二级类码名称映射 |

## 异常处理

| 异常场景 | 前端提示 |
|---------|---------|
| 导入文件格式错误 | "文件格式不合法，请上传.xlsx或.xls文件" |
| 导入文件过大 | "文件过大，请上传不超过200MB的Excel文件" |
| 文件中无有效编码 | "文件中未找到有效的31位测点编码" |
| 导入正在进行中 | "导入任务正在进行中，请等待完成后再次导入" |
| 统计数据为空 | 图表区域显示空状态提示 |
| 清空数据 | 二次确认对话框："确定要清空所有测点数据吗？此操作不可恢复" |

## 验收标准

1. 编码生成统计概览 4 个卡片数值正确
2. 类型维度 4 个卡片统计数字正确
3. 二级类码图表切换风电/光伏/水电显示不同数据
4. 场站图表正确展示各场站编码量
5. 编码生成列表分组展示，展开可见单条编码
6. 列表支持筛选、多选、导出选中、删除选中
7. 全量测点导入选择文件后显示进度条和预估时间
8. 导入过程中终止按钮有效
9. 导入完成后概览卡片数据正确
10. 二级类码和场站图表正确展示
11. 清空数据操作有二次确认，确认后数据清除
12. 各图表在无数据时显示空状态
