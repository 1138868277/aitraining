# 统计分析

## 功能目标

- 支持上传编码稽核数据文件（Excel），将数据导入数据库
- 自动拉取数据库中的编码稽核数据，完成全局统计计算
- 按管理域、场站、类型、校验结果等维度进行细分统计
- 使用柱状图、饼图可视化展示统计数据
- 支持统计数据的导出

## 页面说明

### 页面布局

统计分析页面采用上下布局结构：

- **顶部区域**：数据上传区，包含上传组件和统计概览卡片
- **中部区域**：图表展示区，包含柱状图和饼图
- **底部区域**：统计明细表格

### 页面路径

- `/statistics` — 统计分析主页面

### 页面元素

#### 数据上传区

- 文件拖拽/点击上传区域
- 支持文件类型：.xlsx, .xls
- 文件大小限制：10MB
- 上传后自动触发数据导入和统计刷新
- 显示上传文件名称、导入记录条数、最近上传时间
- 上传过程中显示进度状态

#### 统计概览卡片（4个）

| 卡片名称 | 说明 |
|---------|------|
| 总编码数 | 系统中编码稽核数据总条数 |
| 合规编码数 | 校验结果为合规的编码数量 |
| 异常编码数 | 校验结果为异常的编码数量 |
| 合规率 | 合规编码数 / 总编码数，以百分比展示 |

每个卡片展示数值，合规率卡片同时显示环形进度条。

#### 图表展示区

**柱状图区域：**
- 维度选择器：管理域、场站、类型、校验结果等
- 按选定维度展示各分类的编码数量分布
- 合规/异常双色堆叠显示
- 支持鼠标悬浮查看具体数值

**饼图区域：**
- 维度选择器：管理域、场站、类型等
- 展示选定维度下各分类的占比分布
- 合规/异常用不同颜色区分
- 图例显示各分类名称和占比百分比

#### 统计明细表格

| 字段 | 说明 |
|-----|------|
| 序号 | 行号 |
| 管理域 | 编码所属管理域 |
| 场站 | 编码所属场站 |
| 总编码数 | 该分类下编码总数 |
| 合规数 | 合规编码数量 |
| 异常数 | 异常编码数量 |
| 合规率 | 该分类的合规百分比 |

支持分页展示，默认每页20条。

### 交互说明

1. 上传文件后自动触发数据导入，导入完成后自动刷新统计
2. 切换图表维度时，柱状图和饼图同步更新
3. 柱状图和饼图支持联动：点击柱状图的某个分类，饼图切换至该分类的细分维度

## 接口说明

| 接口名称 | 方法 | 路径 | 说明 |
|---------|------|------|------|
| 上传稽核数据 | POST | /api/statistics/upload | 上传编码稽核数据Excel文件 |
| 获取统计概览 | GET | /api/statistics/overview | 获取系统整体统计概览数据 |
| 按维度统计 | GET | /api/statistics/by-dimension | 按管理域、场站等维度进行细分统计 |
| 获取图表数据 | GET | /api/statistics/chart-data | 获取柱状图、饼图等可视化数据 |
| 获取统计明细 | GET | /api/statistics/details | 分页查询统计明细数据 |
| 导出统计报表 | GET | /api/statistics/export | 导出统计报表为Excel文件 |

## 输入输出

### 上传稽核数据

**请求方式：** multipart/form-data

**请求参数：** file（文件字段）

**输出：**

```json
{
  "fileId": "string",
  "fileName": "string",
  "importCount": 5000,
  "status": "PROCESSING"    // PROCESSING / COMPLETED / FAILED
}
```

### 获取统计概览

**输出：**

```json
{
  "totalCount": 5000,
  "compliantCount": 4200,
  "abnormalCount": 800,
  "complianceRate": 84.0
}
```

### 按维度统计

**输入参数：** dimension（维度类型：managementDomain / station / type / checkResult）

**输出：**

```json
{
  "dimension": "managementDomain",
  "items": [
    {
      "name": "云南区域",
      "totalCount": 2000,
      "compliantCount": 1800,
      "abnormalCount": 200,
      "complianceRate": 90.0
    }
  ]
}
```

### 获取图表数据

**输入参数：** chartType（bar / pie）, dimension（维度类型）

**输出（柱状图）：**

```json
{
  "chartType": "bar",
  "dimension": "station",
  "categories": ["场站A", "场站B", "场站C"],
  "compliantData": [800, 600, 400],
  "abnormalData": [100, 80, 60]
}
```

**输出（饼图）：**

```json
{
  "chartType": "pie",
  "dimension": "managementDomain",
  "series": [
    { "name": "云南区域", "value": 2000, "percentage": 40.0 },
    { "name": "贵州区域", "value": 1500, "percentage": 30.0 },
    { "name": "四川区域", "value": 1500, "percentage": 30.0 }
  ]
}
```

## 业务规则

### 数据导入规则

- 上传的Excel文件必须包含以下字段：编码、编码名称、管理域、场站、类型、校验结果
- 导入时自动去重：同一文件中编码+场站组合重复的只保留第一条
- 导入覆盖策略：新上传的数据覆盖数据库中同一文件来源的旧数据
- 单次导入上限：10000条，超出时提示分批导入

### 统计规则

- 合规率 = 合规编码数 / 总编码数 × 100%，保留1位小数
- 维度统计支持：管理域、场站、类型、校验结果
- 图表默认展示全部数据，可选择维度下钻
- 统计数据实时计算，不缓存

### 导出规则

- 导出字段：管理域、场站、类型、总编码数、合规数、异常数、合规率
- 文件命名规则：`编码统计分析报表_导出时间.xlsx`

## 数据关联

| 关联表 | 关联关系 | 用途 |
|-------|---------|------|
| cec_new_energy_checkdata | 编码稽核数据表 | 拉取稽核数据进行统计分析 |

| cec_new_energy_station_dict | 场站字典 | 按场站维度统计分析 |
| cec_new_energy_type_dict | 类型字典 | 按类型维度统计分析 |

## 异常处理

| 异常场景 | 前端提示 | 处理方式 |
|---------|---------|---------|
| 上传文件格式错误 | "文件格式不合法，请上传.xlsx或.xls文件" | 终止导入 |
| 上传文件超限 | "文件大小超出限制（上限10MB）" | 终止导入 |
| 导入数据为空 | "文件中未检测到有效数据" | 终止导入，建议检查文件内容 |
| 导入字段不匹配 | "文件字段不匹配，请使用标准模板" | 提示下载标准模板 |
| 数据导入超时 | "数据导入超时，请稍后重试或分批导入" | 显示重试按钮 |
| 统计数据为空 | "暂无统计数据，请先上传稽核数据" | 图表区域显示空状态提示 |
| 图表渲染失败 | "图表加载失败，请刷新重试" | 显示重试按钮 |

## 验收标准

1. 上传稽核数据Excel文件后正确导入并显示导入条数
2. 统计概览4个卡片数值正确展示
3. 柱状图按选定维度正确展示编码数量分布
4. 饼图按选定维度正确展示占比分布
5. 切换维度时柱状图和饼图同步更新
6. 柱状图与饼图联动下钻功能正常
7. 统计明细表格分页展示正确
8. 图表支持鼠标悬浮查看具体数值
9. 导出统计报表内容正确、格式规范
10. 重复上传数据覆盖正确
11. 异常场景下正确显示友好提示
