# AItraining

这是北京东方金信科技股份有限公司用于员工 `AI Coding` 培训的脚手架项目。

`AI Agent` 可以根据 [doc/README.md](D:/work/codex/aitraining/doc/README.md:1) 及其子目录中的需求、设计和规范，直接生成 Demo 项目代码。

## 目录说明

- `doc/`：存放项目文档、开发规范和用于培训的目录结构说明
- `code/`：存放项目代码

## 使用方式

1. 先阅读 [doc/01-AI入口.md](D:/work/codex/aitraining/doc/01-AI入口.md:1)
2. 按顺序阅读 `doc` 下的需求、架构、功能、测试、参考和规范文档
3. 基于 `doc` 中的内容生成或完善 `code/` 下的项目代码

## 适用场景

- 学员基于文档手工完成 Demo 项目开发
- AI 基于文档自动生成或协助完善 Demo 项目
